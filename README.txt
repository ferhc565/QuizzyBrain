QuizzyBrain - Versión Web (Estilo Dark)

Esta carpeta contiene la versión web de QuizzyBrain lista para subir a GitHub Pages
o empaquetar como PWA / APK.

Archivos principales:
- index.html
- style.css
- app.js
- manifest.webmanifest
- sw.js
- preguntas.json (banco de preguntas)
- colores.json (colores por categoría)
- carpeta /icons con iconos 192x192 y 512x512

Pantallas:
- Selección de categoría (usa colores de colores.json)
- Quiz por categoría con temporizador por pregunta
- Resultados
- Botón de Configuración (ajusta tiempo por pregunta, guardado en localStorage)

Para usar en GitHub Pages, sube TODOS estos archivos a la raíz del repositorio.
