Quiz Interactivo Web (PWA) - Versión con Configuración y Administración
=====================================================================

Replica la estructura de la app de escritorio:
- Botón / pestaña de CONFIGURACIÓN
- Botón / pestaña de ADMINISTRAR PREGUNTAS
- Pantalla de JUGAR por categorías

Cómo probar
-----------

1. Descomprime el ZIP en una carpeta.
2. En esa carpeta abre una terminal y ejecuta:

   En Windows:
     py -m http.server 8000

3. Abre en el navegador:
   http://localhost:8000

Usa la barra superior para cambiar entre:
- Jugar
- Configuración
- Administrar preguntas
